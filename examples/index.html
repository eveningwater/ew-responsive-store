示例#

本节提供了在不同场景中使用 ew-responsive-store 的实用示例。

基础计数器示例#

一个简单的计数器，其值在 localStorage 中持久化：

1import { useStorage } from "ew-responsive-store";
2
3// 初始化一个计数器，默认值为 0
4const counter = useStorage("counter", 0);
5
6// 增加计数器
7function increment() {
8  counter.value++;
9}
10
11// 减少计数器
12function decrement() {
13  counter.value--;
14}
15
16// 重置计数器
17function reset() {
18  counter.value = 0;
19}

用户偏好设置#

管理在会话之间持久化的用户偏好设置：

1import { useStorage } from "ew-responsive-store";
2import { StoreType } from "ew-responsive-store/typings/core/enum";
3
4// 初始化用户偏好设置，设置默认值
5const preferences = useStorage("userPreferences", {
6  theme: "light",
7  fontSize: "medium",
8  notifications: true,
9});
10
11// 在亮色和暗色主题之间切换
12function toggleTheme() {
13  preferences.value.theme =
14    preferences.value.theme === "light" ? "dark" : "light";
15}
16
17// 更改字体大小
18function setFontSize(size) {
19  preferences.value.fontSize = size; // 'small', 'medium', 或 'large'
20}
21
22// 切换通知
23function toggleNotifications() {
24  preferences.value.notifications = !preferences.value.notifications;
25}

购物车#

实现在页面刷新之间持久化商品的购物车：

1import { useStorage } from "ew-responsive-store";
2
3// 初始化一个空购物车
4const cart = useStorage("shoppingCart", []);
5
6// 向购物车添加商品
7function addToCart(product) {
8  const existingItem = cart.value.find((item) => item.id === product.id);
9
10  if (existingItem) {
11    // 如果商品已存在，增加其数量
12    existingItem.quantity += 1;
13  } else {
14    // 否则，添加新商品，数量为 1
15    cart.value.push({
16      ...product,
17      quantity: 1,
18    });
19  }
20}
21
22// 从购物车移除商品
23function removeFromCart(productId) {
24  const index = cart.value.findIndex((item) => item.id === productId);
25
26  if (index !== -1) {
27    cart.value.splice(index, 1);
28  }
29}
30
31// 更新商品数量
32function updateQuantity(productId, quantity) {
33  const item = cart.value.find((item) => item.id === productId);
34
35  if (item) {
36    item.quantity = quantity;
37  }
38}
39
40// 清空整个购物车
41function clearCart() {
42  cart.value = [];
43}
44
45// 计算购物车中商品的总价
46function getCartTotal() {
47  return cart.value.reduce(
48    (total, item) => total + item.price * item.quantity,
49    0
50  );
51}

表单状态持久化#

保存表单状态，防止在导航离开时数据丢失：

1import { useStorage } from "ew-responsive-store";
2import { StoreType } from "ew-responsive-store/typings/core/enum";
3
4// 使用 sessionStorage 存储临时表单数据
5const formData = useStorage(
6  "contactForm",
7  {
8    name: "",
9    email: "",
10    message: "",
11  },
12  { storage: StoreType.SESSION }
13);
14
15// 更新表单字段
16function updateField(field, value) {
17  formData.value[field] = value;
18}
19
20// 提交表单
21function submitForm() {
22  // 将数据发送到 API
23  sendToApi(formData.value);
24
25  // 成功提交后清空表单
26  formData.value = {
27    name: "",
28    email: "",
29    message: "",
30  };
31}
32
33// 检查表单是否有数据
34function hasFormData() {
35  return (
36    formData.value.name !== "" ||
37    formData.value.email !== "" ||
38    formData.value.message !== ""
39  );
40}

与 React 一起使用#

使用 useReactStorage 的基本 React 示例#

在 React 中使用 ew-responsive-store 的推荐方式是通过 useReactStorage 钩子：

1import React from "react";
2import { useReactStorage, StoreType } from "ew-responsive-store";
3
4function ThemeSwitcher() {
5  const [theme, setTheme] = useReactStorage("theme", "light");
6  const [count, setCount] = useReactStorage("count", 0);
7
8  const toggleTheme = () => {
9    setTheme(theme === "light" ? "dark" : "light");
10  };
11
12  const increment = () => setCount(count + 1);
13  const decrement = () => setCount(count - 1);
14  const reset = () => setCount(0);
15
16  return (
17    <div className={`app ${theme}`}>
18      <h2>主题切换器</h2>
19      <button onClick={toggleTheme}>切换主题</button>
20      <p>当前主题: {theme}</p>
21      
22      <h2>计数器</h2>
23      <div>
24        <button onClick={decrement}>-</button>
25        <span>{count}</span>
26        <button onClick={increment}>+</button>
27        <button onClick={reset}>重置</button>
28      </div>
29    </div>
30  );
31}
32
33export default ThemeSwitcher;

React 待办事项列表#

使用 useReactStorage 的完整待办事项列表示例：

1import React, { useState } from "react";
2import { useReactStorage, StoreType } from "ew-responsive-store";
3
4interface Todo {
5  id: number;
6  text: string;
7  completed: boolean;
8}
9
10function TodoApp() {
11  const [todos, setTodos] = useReactStorage<Todo[]>("todos", []);
12  const [inputValue, setInputValue] = useState("");
13
14  const addTodo = () => {
15    if (inputValue.trim()) {
16      const newTodo: Todo = {
17        id: Date.now(),
18        text: inputValue,
19        completed: false,
20      };
21      setTodos([...todos, newTodo]);
22      setInputValue("");
23    }
24  };
25
26  const toggleTodo = (id: number) => {
27    setTodos(
28      todos.map((todo) =>
29        todo.id === id ? { ...todo, completed: !todo.completed } : todo
30      )
31    );
32  };
33
34  const deleteTodo = (id: number) => {
35    setTodos(todos.filter((todo) => todo.id !== id));
36  };
37
38  const clearCompleted = () => {
39    setTodos(todos.filter((todo) => !todo.completed));
40  };
41
42  return (
43    <div>
44      <h1>待办事项列表</h1>
45      
46      <div>
47        <input
48          type="text"
49          value={inputValue}
50          onChange={(e) => setInputValue(e.target.value)}
51          onKeyPress={(e) => e.key === "Enter" && addTodo()}
52          placeholder="添加新的待办事项..."
53        />
54        <button onClick={addTodo}>添加</button>
55      </div>
56
57      <div>
58        <button onClick={clearCompleted}>清除已完成</button>
59        <p>
60          {todos.filter((t) => !t.completed).length} 个待完成，{" "}
61          {todos.filter((t) => t.completed).length} 个已完成
62        </p>
63      </div>
64
65      <ul>
66        {todos.map((todo) => (
67          <li key={todo.id}>
68            <input
69              type="checkbox"
70              checked={todo.completed}
71              onChange={() => toggleTodo(todo.id)}
72            />
73            <span
74              style={{
75                textDecoration: todo.completed ? "line-through" : "none",
76              }}
77            >
78              {todo.text}
79            </span>
80            <button onClick={() => deleteTodo(todo.id)}>删除</button>
81          </li>
82        ))}
83      </ul>
84    </div>
85  );
86}
87
88export default TodoApp;

跨标签页同步#

React 组件会自动在不同浏览器标签页之间同步：

1import React from "react";
2import { useReactStorage } from "ew-responsive-store";
3
4function MultiTabCounter() {
5  const [count, setCount] = useReactStorage("sharedCounter", 0);
6
7  return (
8    <div>
9      <h2>共享计数器</h2>
10      <p>这个计数器在所有浏览器标签页中同步！</p>
11      <p>计数: {count}</p>
12      <button onClick={() => setCount(count + 1)}>增加</button>
13      <button onClick={() => setCount(count - 1)}>减少</button>
14      <button onClick={() => setCount(0)}>重置</button>
15    </div>
16  );
17}
18
19export default MultiTabCounter;

与 Vue 一起使用（传统方式）#

如果您需要在 React 中使用 Vue 版本（不推荐），可以这样做：

1import React, { useEffect, useState } from "react";
2import { useStorage } from "ew-responsive-store";
3
4function ThemeSwitcher() {
5  // 初始化存储
6  const preferences = useStorage("preferences", { theme: "light" });
7
8  // 本地状态触发重新渲染
9  const [theme, setTheme] = useState(preferences.value.theme);
10
11  // 当存储变化时更新本地状态
12  useEffect(() => {
13    const updateTheme = () => {
14      setTheme(preferences.value.theme);
15    };
16
17    // 初始更新
18    updateTheme();
19
20    // 订阅变化
21    const unsubscribe = preferences.subscribe(updateTheme);
22
23    // 清理
24    return () => {
25      unsubscribe();
26    };
27  }, []);
28
29  const toggleTheme = () => {
30    preferences.value.theme =
31      preferences.value.theme === "light" ? "dark" : "light";
32  };
33
34  return (
35    <div>
36      <h2>主题切换器</h2>
37      <button onClick={toggleTheme}>切换主题</button>
38      <p>当前主题: {theme}</p>
39    </div>
40  );
41}
42
43export default ThemeSwitcher;

高级用法：自定义序列化#

为复杂数据结构使用自定义序列化：

1import { useStorage, parseStr } from "ew-responsive-store";
2import { parseStrType } from "ew-responsive-store/typings/core/enum";
3
4// 为 Date 对象自定义序列化
5const lastVisit = useStorage("lastVisit", new Date(), {
6  serializer: (date) => date.toISOString(),
7  deserializer: (dateStr) => new Date(dateStr),
8});
9
10// 将最后访问时间更新为现在
11function updateLastVisit() {
12  lastVisit.value = new Date();
13}
14
15// 获取最后访问时间的格式化字符串
16function getFormattedLastVisit() {
17  return lastVisit.value.toLocaleString();
18}
